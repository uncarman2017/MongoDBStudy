## 1.3 独特功能
> * 索引 <br/>
    MongoDB支持通用的二级索引，允许多种快速查询，支持唯一索引、复合索引、地理空间索引、全文索引。
> * 聚合 <br/>
    MongoDB支持聚合管道。
> * 特殊的集合类型 <br/>
    支持存在时间有限的集合，适用于那些将在某个时刻过期的数据，如会话(session)。类似，MongoDB也支持固定大小的集合用于保存近期数据，如日志。
> * 文件存储<br/>
    支持大文件和文件元数据存储
> * 不支持功能
    关系数据库的join和复杂的多行事务
    
## 1.4 性能
> * 动态填充
> * 预分配数据文件以利用额外的空间来换取稳定的性能
> * 将尽可能多的内存用作缓存
